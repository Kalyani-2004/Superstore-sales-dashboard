# Superstore Sales Dashboard (Power BI)

## Project Description
This is a Power BI dashboard created using the Superstore dataset. The dashboard helps to understand sales, profit, and business performance using simple visualizations.

## Key Highlights
- Total Sales: 1.6M
- Total Profit: 175K
- Total Quantity: 22K
- Average Ship Days: 4

## Dashboard Insights
- Sales by Category (Technology, Furniture, Office Supplies)
- Sales by Sub-Category (Phones, Chairs, Binders)
- Sales by Region (Central, East, South, West)
- Monthly Sales Trend (2019 & 2020)
- Monthly Profit Trend
- Sales by Segment (Consumer, Corporate, Home Office)
- Sales by Payment Mode (COD, Online, Cards)
